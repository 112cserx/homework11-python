# Python для аналитиков ИБ: поиск негативных событий

## Описание проекта
В рамках домашнего задания выполнен анализ логов информационной безопасности с использованием Python.

Цель работы — выявление подозрительных событий в логах Windows (WinEventLog) и DNS-запросах, а также их визуализация.

---

## Используемые технологии

- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code
- GitHub

---

## Этапы выполнения

### 1. Загрузка и подготовка данных
- Загружен набор данных botsv1
- Выполнена нормализация JSON-структуры
- Данные преобразованы в DataFrame

### 2. Анализ данных

#### WinEventLog
Выявлены подозрительные события по EventCode:
- 4624 — успешный вход
- 4688 — создание процесса
- 4689 — завершение процесса
- 4703 — изменение привилегий

#### DNS-логи
Проанализированы DNS-запросы:
- обнаружены подозрительные домены
- выявлены потенциальные C2-запросы

### 3. Визуализация
Построены графики:
- Top подозрительных WinEventLog событий
- DNS-запросы
- Объединённая визуализация событий
